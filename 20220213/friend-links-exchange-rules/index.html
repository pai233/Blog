本站友链交换规则

发表于2022-02-13|更新于2022-03-20|咕咕小站博客动态
|字数总计:633|阅读时长:1分钟|阅读量:

首先，感谢您愿意交换友链！但交换友链还有以下规则，还请您详细阅读！

域名

对于一个网站来说，域名相当于它的门户。所以，一个好的域名非常重要。因此，博主拒绝以下类型的域名交换友链：

  1. 使用免费域名的
  2. 域名中有违反中华人民共和国法律的字眼的
  3. .onion等无法通过正常互联网访问的
  4. 其他此处未考虑到，但经审核后拒绝的情节

免费域名

免费域名包括但不限于以下几种：

  1. 由Freenom公司所运营的.tk、.cf、.ga、.ml等免费域名
  2. 由 Joshua Anderson 运营的 Afraid FreeDNS 提供的免费子域名
  3. 其它不包括在PUBLIC SUFFIX LIST中的免费子域名服务

请注意，由于国内的部分Pages服务，如Gitee Pages（gitee.io）拒绝提交其域名至PUBLIC SUFFIX LIST，根据第三条的定义，此类申请将被拒绝。我们建议您将博客部署到其他平台，如Cloudflare Pages、Vercel等。

免费域名不包括以下定义：

  1. 在Freenom付费购买的.tk等域名（需提供凭证）
  2. Github Pages服务的免费子域名（github.io）
  3. Cloudflare Pages服务的免费子域名（pages.dev）
  4. 其他已纳入PUBLIC SUFFIX LIST的免费子域名服务

内容

对于一个博客，优质内容也是很重要的一个关键。请保证您的博客有 实质性 内容，博主拒绝站点内容多为采集、抄袭的站点交换友链。博客可以长草，但不能滥竽充数。

另外，如果您的网站出现冒用备案、传播违反中华人民共和国法律的内容时，博主将拒绝此类友链并举报至相关部门。

网络

由于本站主要向中国大陆地区用户，因此，您需要保证您的博客在中国大陆地区访问正常，未出现css样式丢失，大量图片404等严重影响用户体验的情况。
同时，为了保证访问的安全性，您的博客需要开启https。目前多数主流平台均以支持一键开启https，您可以搜索🔍您的平台如何开启https。

FAQ

  1. 我的免费域名已经用很久了，怎么办？
    如果您正在使用免费域名，我们会审核您的文章内容情况，如果文章内容优质且均为原创，博主依旧会添加您的友链。
